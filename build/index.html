DevOps
Comparator

This website consists of multiple comparators, each for a component needed to have a complete DevOps ecosystem.

Each comparator gathers information about the most popular services, obtained by manual research and without the help of crawlers.

If for instance you are interested in using CI/CD to create DevOps pipelines, click on the CI/CD configurator tab and enter your parameters, in order to decrease the number of results and reach a more suitable recommendation.

Unlike other comparators, this one calculates the expected prices on the fly based on the parameters entered.

The parameters that are possible to choose are also particular to this configurator and have been chosen after gathering information on a large number of services, cloud and self-hosted.

Currently, there are comparators for version control, continuous integration and deployment and team chat platforms.

It is also possible to view a set of graphs related to the information gathered on each type of comparator.

This project was carried out within the scope of Afonso Figuinha's MSc thesis in partnership with UNL-SST, which ended in March 2023.

For more information about this project and how to update information, see the wiki of this github repository.
